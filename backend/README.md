#### Description

This service exposes a RESTful API endpoint that accepts binary encoded data
inputs and returns human-readable representations in response.

In the backend project directory:

### Installation

```bash
$ yarn
```

### Running the app

```
$ yarn start:dev
```

### Test

```
$ npm run test

```
